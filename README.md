# t-SNE Plot with Gene Labels

Create a Virtual Environment
```bash
python -m venv venv_dash
source venv_dash/bin/activate  
```

Install Dependencies
```bash
pip install -r requirements.txt
```

How to Run the App:
```bash
python main.py
```

Open your browser and go to:
```arduino
http://127.0.0.1:8050
```
